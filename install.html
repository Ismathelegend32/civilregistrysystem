Waxaad isticmaalaysaa App-ka (WebView)

Haddii aad telefoon kale ku rakibaysid, fur link-ga Chrome ama Safari, kadib raac tilmaamaha hoose.

Android — Ku rakib Home Screen

  1. Guji badhanka Install App hoose (haddii uu muuqdo).
  2. Ama menu-ga browser-ka () → Add to Home screen / Install app.

iPhone — Ku rakib Home Screen

  1. Fur boggan Safari (ma aha Chrome gudaha iPhone).
  2. Guji Share
  3. Dooro Add to Home Screen
  4. Guji Add

Ku rakib App-ka

Si app-ku u noqdo sida telefoonkaaga app kale, ku dar Home Screen ka hor intaadan galin.

Markaad Home Screen ku darto, mar dambe si toos ah ayuu u furmi doonaa.
